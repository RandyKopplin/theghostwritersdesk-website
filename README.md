# theghostwritersdesk.com

Static site for **The Ghostwriter's Desk** — done-for-you delivery of the
writing-heavy assets architects use to win better clients.

## Pages

- `index.html` — main service landing page
- `client-primer-build/index.html` — The Client Primer Build (5-day educational
  email sequence, $1,500–$5,000 done-for-you)

## Deploy

Hosted on Vercel. The `master` branch deploys to production at
`theghostwritersdesk.com`. The repo is the canonical source of truth for the
domain.

## History

Migrated from `randykopplin.com/theghostwritersdesk/` on 2026-05-26 when The
Ghostwriter's Desk consolidated as a standalone brand and primary offer.
The previous URLs 301-redirect from `randykopplin-website`'s `vercel.json`.

## Brand

Uses the AEC-orange palette (#f79546 accent on dark backgrounds) and
Barlow + DM Sans + JetBrains Mono typography, matching the design system
established on randykopplin.com and aeclogix.com. See
`@secondbrain/AECLogix/Brand/visual-identity.md` for the canonical brand spec.
